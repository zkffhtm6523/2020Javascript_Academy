연산자

6.조건선택 연산자(삼항 연산자) : (? : )을 쌍으로 사용하는 연산자
- 조건에 대해서 true일 때와 false일 때 다르게 실행하도록 하는 연산자//2. 조건선택 연산자를 활용
- ? 앞은 조건, : 앞은 조건이 참일 때, : 뒤는 조건이 거짓일 때
- if문의 대용으로 사용, 간단한 if~else의 대용으로 사용
- 복잡한 조건문일 때는 if문을 사용하기를 권장
단항 : ++a, --a, !a
이항 : 10 + 20
삼항 : 조건 ? 참 : 거짓

연산자 우선순위 정리

1순위 : 단항(++,--,!) 2순위 : 산술연산자(+,-,*,/,%),(*,/,%)가 (+,-)보다 우선순위가 높다 3순위 : 비교연산자(>, >=, <, <=) >(==, !=) > (===, !==) 4순위 : 논리연산자(&&, ||) - &&가 ||보다 우선순위가 높다. 5순위 : 조건선택연산자(? : ) 6순위 : 대입연산자, 복합대입(할당)연산자 : =, +=, -=, *=, /+, %=
