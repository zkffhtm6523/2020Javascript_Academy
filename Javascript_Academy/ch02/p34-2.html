자바스크립트를 작성하는 규칙

1. 대소문자를 구분해서 작성한다.
2. 들여쓰기를 해서 작성한다.
3. 행의 끝은 세미콜론으로 마무리한다.
4. 주석을 사용하여 설명을 작성한다.
5. 식별자(identifier)는 정해진 규칙을 지켜 작성한다.
ex) 변수, 함수, 속성, 클래스 ...
6. 예약어는 식별자로 사용할 수 없다.

변수

변수 : 값(데이터)를 저장하는 공간

변수를 선언하는 규칙

1. 이름을 의미있게 짓는다.(권장) 2. 변수명은 낙타 모양으로 만든다.(Camel Case)(권장)
3. 변수명은 대소문자를 구분한다.
4. 변수명의 첫글자는 영어 대소문자,_,$를 사용하고, 이후로는 숫자도 사용할 수 있다.
- $(달러)는 사용하지 않기를 권장 : 시스템 변수들이 $가 붙어 있으므로 시스템 변수와의 구분을 위해서 사용하지 않는 것이 좋겠다.
5. 예약어는 변수이름으로 사용불가하다.
